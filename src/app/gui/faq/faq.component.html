FAQ

This section helps you to find answers to questions that were asked before. If you can't find an answer you can use the bug report to ask us.

What are the system requirements to use OCTRA?

To make sure, that OCTRA works well it is recommended to use Chrome, because OCTRA was tested with this Browser. It could be possible that OCTRA works with the latest versions of Firefox and Opera, but it hasn't been tested before. Safari is not supported.
Recommended system configuration:
  1. browser: Google Chrome
  2. min. 2GB RAM
  3. min 128MB GPU
  4. connection to the internet
If you want to use OCTRA on a PC or notebook which is less than 5 years old, this should be enough for OCTRA to work.

The progress bar stopped and OCTRA does not continue

Please make sure that you don't use Safari, because it is not supported. If error messages have been appeared these could help to find the source of this error. You can try to click on 'Retry'. If this still does not work, you should do a full refresh (in Chrome):
  1. Do a right-click on any place on the page and click on 'Inspect':
  2. Now a new developement window appears
  3. Now you can click (right-click) on the reload icon before the address bar. On the context menu click on 'Clear cache and make full refresh'

If this still does not work, please send us a bug report
